This repository contains my notes, exercises, practice problems, and projects completed while studying Computer Science through the OSSU curriculum.

The goal is not just to finish the curriculum, but to build a strong foundation in:

## Why I'm Doing This

I believe the best way to learn Computer Science is through consistent daily practice.

This repository serves as:

* My learning journal
* A record of my progress
* A collection of notes for future review
* Proof of consistent effort and growth

## Learning Philosophy

* Learn every concept deeply.
* Write code daily.
* Solve exercises before looking at solutions.
* Focus on understanding, not memorization.
* Build strong fundamentals before moving to advanced topics.
