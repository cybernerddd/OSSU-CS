# 🏦 Bank Account System (OOP)

A banking system built in Python using Object-Oriented Programming.

This project was created after completing Lecture 20 of MIT's **6.100L - Introduction to Computer Science and Programming Using Python**.

Re-enforced my knowledge with building this.

---

## Features

### Base Account

- Deposit
- Withdraw
- Transfer
- Transaction History
- Account Status
- Equality Comparison
- String Representation

---

### Savings Account

- Interest Rate
- Apply Interest
- Daily Withdrawal Limit

---

### Checking Account

- Overdraft Support
- Remaining Overdraft Tracking
- Monthly Overdraft Reset

---

### Business Account

- Business Name
- Business ID
- Business-specific Account Type

---

## Helper Functions

- Total money across accounts
- Richest account
- Active accounts only
